Inline CSS

Inline CSS adalah gaya yang diterapkan langsung pada elemen HTML dengan menggunakan atribut style.

Contoh:

<h1 style="color:blue;">This is a Blue Heading</h1>

Internal CSS

Internal CSS ditulis di dalam elemen <style> di bagian <head> dari halaman HTML.

Contoh:

<style>
body { background-color: powderblue; }
h1 { color: blue; }
p { color: red; }
</style>

External CSS

External CSS adalah file terpisah yang dihubungkan ke halaman HTML melalui elemen <link>.

Contoh:

<link rel="stylesheet" href="styles.css">
